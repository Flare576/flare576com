# Concept

As I pursue my journey of both determining if I enjoy painting stuff and of improving at it, I'm going to be doing milestone checks with various models/characters that I'm familiar with of "Increasing difficulty," as determined by the super scientific process of "Ask a friend how hard it might be to paint."

I didn't ask anyone how hard these things would be to _print_, though, so that's going to be another _tangential_ journey.

I think the process here is going to be that I create one post per milestone, then use it as a "Draft" until it's completed. This is because, based on the Chocobo, it's not going to be a quick process.
