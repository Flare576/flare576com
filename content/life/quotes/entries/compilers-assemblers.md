# Learning to Code

My son was asking some questions about the different programming languages I know, and I mentioned that some were faster than others, and some were more portable than others.

Which lead to him asking why that was, obviously, so I ended up talking about compiled languages and machine code and then he stopped me.

"So, you're saying that a long time ago people built compilers and assemblers, and now everyone just uses them and very few people would know exactly how to fix something?" he asked.

"Well, yeah, I guess that's pretty much true..." I admitted, knowing that it's not magic or anything but...

"Ok, so basically, Assemblers are like that robot in Demon Slayer - old, complex, and no one really knows how to fix 'em if someone hits them with a sword."

I didn't reply. I didn't need to.
